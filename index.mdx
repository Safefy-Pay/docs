Safefy é uma plataforma de pagamentos focada em **PIX**, com autenticação segura, webhooks confiáveis e ferramentas para conciliação. A API foi desenhada para ser simples, previsível e segura.

## Fluxo principal

1. **Crie suas credenciais** no painel Safefy (ambiente Sandbox ou Produção)
2. **Gere um token de acesso** via OAuth2 (client credentials)
3. **Crie clientes** (opcional) para organizar pagamentos
4. **Crie transações** e receba o QR Code / Copia e Cola
5. **Receba webhooks** com o status do pagamento
6. **Consulte transações e saldos** para conciliação

## Ambientes

- **Sandbox**: testes e simulações de pagamento
- **Produção**: transações reais

## Versão e base path

Todas as rotas da API usam o prefixo **/v1**.

## Valores monetários

Todos os valores são enviados em **centavos** (menor unidade da moeda).

| Valor real | Valor na API |
|------------|--------------|
| R$ 1,00    | 100          |
| R$ 10,50   | 1050         |
| R$ 100,00  | 10000        |

## Segurança

- Tokens JWT com expiração
- Validação de IP (quando configurada)
- Assinatura de webhook em `X-Safefy-Signature`
