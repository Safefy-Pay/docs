## Endpoint de health check

Você também pode monitorar a API programaticamente:

```
GET https://api-payment.safefypay.com.br/health
```

- **HTTP 200** — API saudável e operacional
- **Diferente de 200** — API indisponível ou degradada

---

## Quando usar

- **Monitoramento externo** — integre o health endpoint ao seu sistema de alertas (ex: UptimeRobot, Datadog)
- **Diagnóstico de incidentes** — verifique rapidamente se o problema está no nosso lado ou no seu
- **Ambiente de staging** — confirme que a API está acessível antes de rodar testes automatizados

<Note>
Para histórico de incidentes, janelas de manutenção e atualizações de status em tempo real, consulte **https://status.safefypay.com.br**.
