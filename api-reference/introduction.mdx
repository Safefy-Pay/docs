---
title: "Introdução da API"
description: "Guia rapido para usar a API Reference"
---

Use esta secao para testar endpoints no `Try It`.

Fluxo minimo:
1. Gere um token em [Autenticação](/authentication)
2. Abra a rota desejada em API Reference
3. Envie `Authorization: Bearer <token>`

Ambientes:
- `Sandbox` para testes
- `Production` para operação real
