Retorna o saldo atual da sua conta Safefy, incluindo valores disponíveis, pendentes e totais acumulados.

## Entendendo os Valores

| Campo | Descrição |
|-------|-----------|  
| `balance.available` | Saldo disponível para saque agora |
| `balance.withdrawNowAvailable` | Valor disponível para saque neste momento |
| `balance.requiresFullWithdrawalNow` | Quando `true`, o próximo saque precisa ser no valor exato de `withdrawNowAvailable` |
| `balance.pending` | Pagamentos aguardando confirmação |
| `balance.reserved` | Valores reservados para saques em processamento |
| `balance.total` | Soma de available + pending + reserved |

<Note>
Todos os valores monetários estão em **centavos**. Para converter para reais, divida por 100.
</Note>

<Warning>
Para criar saques, use `withdrawNowAvailable` como referência principal. Se `requiresFullWithdrawalNow` vier como `true`, faça o saque exatamente nesse valor para continuar o fluxo normalmente.
</Warning>
