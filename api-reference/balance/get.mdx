---
title: "Consultar saldo"
description: "Consulte o saldo da sua conta"
openapi: "api-reference/openapi.json GET /v1/balance"
---

Retorna o saldo atual da sua conta Safefy, incluindo valores disponíveis, pendentes e totais acumulados.

## Entendendo os Valores

| Campo | Descrição |
|-------|-----------|  
| `balance.available` | Saldo disponível para saque |
| `balance.pending` | Pagamentos aguardando confirmação |
| `balance.reserved` | Valores reservados para saques em processamento |
| `balance.total` | Soma de available + pending + reserved |

<Note>
Todos os valores monetários estão em **centavos**. Para converter para reais, divida por 100.
</Note>
