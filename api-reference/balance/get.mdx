---
title: "Consultar saldo"
description: "Consulte o saldo da sua conta"
openapi: "/api-reference/openapi.json GET /v1/balance"
---

Retorna o saldo atual da sua conta Safefy, incluindo valores disponíveis, pendentes e totais acumulados.

Todos os valores monetários são retornados em **centavos**.

<Warning>
Para criar saques, use `withdrawNowAvailable` como referência principal. Se `requiresFullWithdrawalNow` vier como `true`, faça o saque exatamente nesse valor para continuar o fluxo normalmente.
</Warning>
