---
title: "Webhooks de Saque"
description: "Eventos de webhook enviados para mudancas de status de saque"
---

A Safefy envia webhook para o `callbackUrl` quando o saque muda para status final.

Eventos enviados:
- `cashout.completed`
- `cashout.failed`
- `cashout.rejected`
- `cashout.cancelled`

Headers principais:
- `X-Safefy-Signature`
- `X-Safefy-Event`
- `X-Safefy-Delivery`
- `X-Safefy-Attempt`

A entrega usa ate 3 tentativas com backoff. Retorne `2xx` para confirmar recebimento.
