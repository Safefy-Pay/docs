---
title: "Cancelar saque"
description: "Cancele um saque pendente"
openapi: "/api-reference/openapi.json POST /v1/cashouts/{id}/cancel"
---

Cancela um saque que ainda esta em status `Pending`. Saques em processamento ou concluidos nao podem ser cancelados.

## Parametros de Path

<ParamField path="id" type="string" required>
  ID do saque (UUID).
</ParamField>

<Warning>
Apenas saques com status `Pending` podem ser cancelados. O valor reservado sera devolvido ao saldo disponivel.
</Warning>
