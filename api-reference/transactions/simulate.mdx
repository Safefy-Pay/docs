---
title: "Simular transação"
description: "Simule mudanças de status no Sandbox"
openapi: "/api-reference/openapi.json POST /v1/transactions/{transactionId}/simulate"
---

Simula ações em uma transação no ambiente Sandbox. Use para testar fluxos de pagamento sem dinheiro real.

<Warning>
Disponível apenas no ambiente **Sandbox**.
</Warning>

## Parâmetros de Path

<ParamField path="transactionId" type="string" required>
  ID da transação (UUID).
</ParamField>

## Parâmetros de Body

<ParamField body="action" type="string" required>
  Ação a simular:
  - `pay` - Simula o pagamento (muda status para Completed)
  - `expire` - Simula expiração (muda status para Expired)
  - `fail` - Simula falha (muda status para Failed)
  - `refund` - Simula estorno (muda status para Refunded)
</ParamField>

<Note>
A simulação dispara webhooks normalmente, permitindo testar seu endpoint de callback.
</Note>
