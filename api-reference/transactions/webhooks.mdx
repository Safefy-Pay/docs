---
title: "Webhooks de transação"
description: "Eventos de webhook enviados para mudanças de status da transação"
---

A Safefy envia webhook para o `callbackUrl` quando o status da transação muda.

Eventos principais:
- `payment.completed`
- `payment.expired`
- `payment.failed`
- `payment.cancelled`
- `payment.refunded`
- `payment.partially_refunded`

Headers principais:
- `X-Safefy-Signature`
- `X-Safefy-Event`
- `X-Safefy-Delivery`
- `X-Safefy-Attempt`

A entrega usa até 3 tentativas. Retorne `2xx` para confirmar recebimento.
