## Comece em 4 passos

<Steps>
  <Step title="Crie credenciais">
    No painel Safefy, crie uma credencial de API para o ambiente desejado.
  </Step>
  <Step title="Gere o access token">
    Faça uma requisição para `/v1/auth/token` com `grantType`, `publicKey` e `secretKey`.
  </Step>
  <Step title="Crie a primeira transação">
    Crie um cliente (opcional) e depois gere uma transação PIX para obter o QR Code.
  </Step>
  <Step title="Receba webhooks">
    Configure a `callbackUrl` para receber atualizações de status em tempo real.
  </Step>
</Steps>

## Exemplo rápido (token)

```bash
curl -X POST https://api.safefypay.com.br/v1/auth/token \
  -H "Content-Type: application/json" \
  -d '{
    "grantType": "client_credentials",
    "publicKey": "pub_...",
    "secretKey": "sec_..."
  }'
```

## Exemplo rápido (criar transação PIX)

```bash
curl -X POST https://api.safefypay.com.br/v1/transactions \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "method": "pix",
    "amount": 15500,
    "currency": "BRL",
    "description": "Pedido #456",
    "callbackUrl": "https://seusite.com/webhooks/safefy"
  }'
```

<Tip>
  Em Sandbox, use a simulação de transações para testar o fluxo completo sem pagamentos reais.
</Tip>
