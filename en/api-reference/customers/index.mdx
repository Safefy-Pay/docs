A **Customer** represents a person or company that makes payments through your integration. Customers can be reused across multiple transactions, and their data is pre-filled automatically in payment flows.

## The Customer object

```json
{
  "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "externalId": "user-456",
  "name": "John Smith",
  "email": "john@example.com",
  "document": "12345678901",
  "documentType": "CPF",
  "phone": "+5511999999999",
  "status": "Active",
  "metadata": "{\"plan\": \"premium\"}",
  "address": {
    "street": "Main Street",
    "number": "100",
    "complement": "Suite 5",
    "neighborhood": "Downtown",
    "city": "São Paulo",
    "state": "SP",
    "postalCode": "01001000",
    "country": "BR"
  },
  "createdAt": "2024-01-10T09:00:00Z"
}
```

## Attributes

<ParamField body="id" type="string">
  Unique UUID v4 identifier, auto-generated by Safefy.
</ParamField>

<ParamField body="externalId" type="string">
  Your internal ID for this customer.
</ParamField>

<ParamField body="name" type="string">
  Customer's full name.
</ParamField>

<ParamField body="email" type="string">
  Customer's email address.
</ParamField>

<ParamField body="document" type="string">
  Customer's CPF or CNPJ (digits only, no punctuation).
</ParamField>

<ParamField body="documentType" type="string">
  Document type: `CPF` (individual) or `CNPJ` (company).
</ParamField>

<ParamField body="phone" type="string">
  Customer's phone number in international format. Example: `+5511999999999`.
</ParamField>

<ParamField body="status" type="string">
  Customer status: `Active` or `Inactive`.
</ParamField>

<ParamField body="metadata" type="string">
  JSON string with custom data. Maximum 1000 characters.
</ParamField>

<Expandable title="address">
  <ParamField body="address.street" type="string">Street name.</ParamField>
  <ParamField body="address.number" type="string">Street number.</ParamField>
  <ParamField body="address.complement" type="string">Apartment, suite, or additional address info.</ParamField>
  <ParamField body="address.neighborhood" type="string">Neighborhood or district.</ParamField>
  <ParamField body="address.city" type="string">City.</ParamField>
  <ParamField body="address.state" type="string">State abbreviation (2 letters). Example: `SP`.</ParamField>
  <ParamField body="address.postalCode" type="string">Postal code (digits only). Example: `01001000`.</ParamField>
  <ParamField body="address.country" type="string">Country code. Example: `BR`.</ParamField>
</Expandable>

<ParamField body="createdAt" type="string">
  Date and time when the customer was created, in ISO 8601 format.
</ParamField>
