---
title: "Update customer"
description: "Update an existing customer's information"
openapi: "/api-reference/openapi.json PATCH /v1/customers/{id}"
---

Updates an existing customer. All fields are optional  only provide the fields you want to change.

## Parameters

<ParamField path="id" type="string" required>
  The unique UUID identifier of the customer to update.
</ParamField>

<ParamField body="name" type="string">
  Customer's full name.
</ParamField>

<ParamField body="email" type="string">
  Customer's email address.
</ParamField>

<ParamField body="document" type="string">
  Customer's CPF or CNPJ (digits only, no punctuation).
</ParamField>

<ParamField body="documentType" type="string">
  Document type: `CPF` or `CNPJ`.
</ParamField>

<ParamField body="phone" type="string">
  Customer's phone number in international format. Example: `+5511999999999`.
</ParamField>

<ParamField body="status" type="string">
  Customer status: `Active` or `Inactive`.
</ParamField>

<ParamField body="metadata" type="string">
  JSON string with custom data. Maximum 1000 characters.
</ParamField>
