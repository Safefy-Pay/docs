---
title: "Create cashout"
description: "Request a withdrawal to a PIX key"
openapi: "/api-reference/openapi.json POST /v1/cashouts"
---

Creates a PIX cashout to the informed destination key.

<Warning>
Always use `withdrawNowAvailable` from [`GET /v1/balance`](/en/api-reference/balance/get).
If `requiresFullWithdrawalNow` is `true`, the cashout amount must be exactly `withdrawNowAvailable`.
</Warning>
