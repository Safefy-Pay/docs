---
title: "List cashouts"
description: "Retrieve a paginated list of cashouts"
openapi: "/api-reference/openapi.json GET /v1/cashouts"
---

Returns a paginated list of cashouts for the authenticated merchant.

## Parameters

<ParamField query="page" type="integer">
  Page number. Default: `1`.
</ParamField>

<ParamField query="pageSize" type="integer">
  Number of items per page. Default: `20`. Maximum: `100`.
</ParamField>

<ParamField query="status" type="string">
  Filter by status: `Pending`, `Processing`, `Completed`, `Failed`, `Rejected`, or `Cancelled`.
</ParamField>

<ParamField query="externalId" type="string">
  Filter by your external ID.
</ParamField>

<ParamField query="startDate" type="string">
  Start date filter in ISO 8601 format.
</ParamField>

<ParamField query="endDate" type="string">
  End date filter in ISO 8601 format.
</ParamField>
