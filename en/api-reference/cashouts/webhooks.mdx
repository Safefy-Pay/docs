---
title: "Cashout webhooks"
description: "Webhook events sent for cashout status changes"
---

Safefy sends webhooks to your `callbackUrl` when a cashout reaches a final status.

Main events:
- `cashout.completed`
- `cashout.failed`
- `cashout.rejected`
- `cashout.cancelled`

Main headers:
- `X-Safefy-Signature`
- `X-Safefy-Event`
- `X-Safefy-Delivery`
- `X-Safefy-Attempt`

Delivery uses up to 3 retry attempts. Return `2xx` to confirm receipt.
