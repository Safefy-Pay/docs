---
title: "List transactions"
description: "Retrieve a paginated list of transactions"
openapi: "/api-reference/openapi.json GET /v1/transactions"
---

Returns a paginated list of transactions for the authenticated merchant.

## Parameters

<ParamField query="page" type="integer">
  Page number. Default: `1`.
</ParamField>

<ParamField query="pageSize" type="integer">
  Number of items per page. Default: `20`. Maximum: `100`.
</ParamField>

<ParamField query="method" type="string">
  Filter by payment method: `Pix` or `Boleto`.
</ParamField>

<ParamField query="status" type="string">
  Filter by status: `Pending`, `Completed`, `Expired`, `Failed`, or `Refunded`.
</ParamField>

<ParamField query="externalId" type="string">
  Filter by your external ID.
</ParamField>

<ParamField query="customerId" type="string">
  Filter by customer ID.
</ParamField>

<ParamField query="startDate" type="string">
  Start date filter in ISO 8601 format.
</ParamField>

<ParamField query="endDate" type="string">
  End date filter in ISO 8601 format.
</ParamField>
