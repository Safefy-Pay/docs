---
title: "Create transaction"
description: "Create a new PIX or Boleto charge"
openapi: "/api-reference/openapi.json POST /v1/transactions"
---

Creates a new payment transaction. Supports **PIX** and **Boleto** payment methods.

<Info>
QR Code images are **not returned** by the API. You should generate the image in your frontend using the `pix.copyAndPaste` value.

See the dedicated guide: [Generating PIX QR Code](/en/api-reference/transactions/qr-code).
</Info>
