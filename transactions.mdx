A API de transações unifica os métodos de pagamento. No momento, **PIX** está disponível.

## Endpoints

- **POST** `/v1/transactions`
- **GET** `/v1/transactions`
- **GET** `/v1/transactions/{id}`
- **POST** `/v1/transactions/{id}/simulate` (Sandbox)

## Criar transação PIX

Campos obrigatórios:
- `method` (use `pix`)
- `amount` (em centavos)
- `currency` (use `BRL`)
- `description`

Campos opcionais:
- `externalId`, `customerId`, `callbackUrl`, `metadata`
- `pixExpirationMinutes`, `customerName`, `customerDocument`

```json
{
  "method": "pix",
  "amount": 15500,
  "currency": "BRL",
  "description": "Pagamento do pedido #456",
  "callbackUrl": "https://seusite.com/webhooks/safefy"
}
```

A resposta inclui o QR Code e o Copia e Cola, além de status e datas relevantes.

## Listar e consultar

Use **GET** `/v1/transactions` para listar transações com filtros por `method`, `status`, `externalId`, `customerId`, `startDate` e `endDate`.

Use **GET** `/v1/transactions/{id}` para obter todos os detalhes de uma transação específica.

## Simular transações (Sandbox)

Use **POST** `/v1/transactions/{id}/simulate` com:

```json
{ "action": "complete" }
```

Ações suportadas: `complete`, `expire`, `fail`, `refund`.
